# LUMA MFL KBINs

Constraint-layer instruction system for Luma agents.

This repository defines:

- core scroll system (I–XI)
- DataDiddler constraint extensions (XII–XX)
- integration surfaces
- deployment payloads

This is not a runtime system.

It is a constraint system that governs how agents perceive, accept, and produce structured output.

---

## Structure

- core_scrolls/ → base system
- datadiddler_extension/ → DataDiddler constraints
- integration/ → instruction alignment
- payloads/ → deployment artifacts

---

## Purpose

To enforce structural coherence and prevent invalid system construction at the cognitive layer.

---

## What this is not

- not a DataDiddler runtime repo
- not a lens or kernel
- not executable code

It is a constraint surface for agents.
